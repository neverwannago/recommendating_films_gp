# Movie Recommendation System

## Описание
**Movie Recommendation System** — это небольшое консольное приложение на Python, которое помогает людям находить фильмы, которые им понравятся. Система учитывает ваши любимые жанры, оценки просмотренных фильмов и даже предпочтения других пользователей, чтобы предлагать действительно интересные варианты.  

Проект создавался как учебное задание и одновременно как способ показать, как работают основные принципы **объектно-ориентированного программирования (ООП)**: инкапсуляция, наследование, полиморфизм, абстрактные классы и методы. Кроме того, мы использовали базовые паттерны проектирования, чтобы сделать код удобным и легко расширяемым.

### Цели проекта
Мы сделали эту систему, чтобы:
- На практике освоить принципы ООП и научиться строить модульный код.  
- Предоставить пользователям инструмент, который помогает получать персональные рекомендации фильмов на основе их вкусов и оценок.  
- Сделать систему гибкой и расширяемой, чтобы в будущем можно было добавлять новые алгоритмы и функции без глобальной переработки кода.

### Основные возможности
1. **Управление пользователями**
   - Регистрация и вход в систему.  
   - Сохранение информации о просмотренных фильмах и выставленных оценках.  
   - Настройка любимых жанров для более точных рекомендаций.  

2. **Работа с фильмами**
   - Просмотр всех фильмов, доступных в библиотеке.  
   - Фильтрация по жанрам и рейтингам.  
   - Возможность добавления новых фильмов (для разработчиков).

3. **Алгоритмы рекомендаций**
   - **По жанрам:** система рекомендует фильмы из любимых жанров пользователя.  
   - **По рейтингу:** показывает самые популярные и высоко оцененные фильмы.  
   - **По похожим пользователям:** предлагает фильмы, которые понравились другим пользователям с похожими интересами.

4. **Консольный интерфейс**
   - Простое и понятное меню, позволяющее быстро ориентироваться в приложении.  
   - Легкий доступ ко всем функциям: просмотр, оценка фильмов и получение рекомендаций.

### Преимущества
- Поддержка сразу нескольких стратегий рекомендаций.  
- Возможность гибкой настройки предпочтений пользователя.  
- Модульная структура кода, которая упрощает добавление новых функций и алгоритмов.

## Установка
Для начала работы необходимо склонировать репозиторий и перейти в папку проекта:

```bash
git clone https://github.com/neverwannago/recommendating_films_gp.git
cd recommendating_films_gp
````

## Использование

Запуск приложения:

```bash
python main.py
```

После запуска вы сможете:

* Создать нового пользователя или войти под существующим аккаунтом.
* Просматривать список фильмов и ставить им оценки.
* Получать рекомендации на основе выбранной стратегии.
* Настраивать любимые жанры для более персонализированных советов.

**Пример действий пользователя:**

1. Войти в систему или зарегистрироваться.
2. Просмотреть список доступных фильмов.
3. Оценить просмотренные фильмы.
4. Получить рекомендации на основе выбранной стратегии.

## Структура проекта
Проект организован так, чтобы код было легко читать и расширять. Вот основные папки и файлы:

```

/recommendating_films_gp/
│
├─ data/
│   └─ data.py             # Основные классы и сущности: Movie, User, Genre
|   └─ data_manager/
│      └─ data_manager.py     # Менеджер работы с данными: добавление, хранение и загрузка
│
├─ recommendations/
│   └─ recommendations.py  # Алгоритмы рекомендаций: по жанрам, по рейтингу, по похожим пользователям
│
├─ gui/
│   └─ gui.py              # Консольный интерфейс приложения
│
└─ README.md               # Описание проекта

```
